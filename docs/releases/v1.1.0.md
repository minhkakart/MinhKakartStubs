# v1.1.0

This release adds the first runnable ASP.NET Core sample app and expands the runtime registration story with production-oriented hardening.

## Added

- New `DiDecoration.Sample` ASP.NET Core app that demonstrates:
  - service registration via `RegisterDecorators(...)`
  - hosted-service registration
  - typed HTTP client registration
  - options binding from configuration
- Keyed service registration support through `ServiceAttribute.Key`
- Keyed hosted-service resolution through `BackgroundServiceAttribute.Key`
- Fail-fast validation for hosted-service lifetime mismatches

## Documentation

- Updated the root `README.md` with sample-app and release-note links
- Added versioned release notes under `docs/releases/`

## Notes

The sample app is intentionally small and is meant as a reference starting point. Run it with:

```powershell
dotnet run --project .\DiDecoration.Sample\DiDecoration.Sample.csproj
```
